Detailed Description

Subroutine for setting Type_Conservative variable.

It is a generic interface to 5 different subroutines as it can be used for setting scalar variables, 1D/2D/3D or 4D arrays. The calling signatures are: 

 ...
 integer(I_P):: Ns
 real(R_P):: rs(1:Ns)
 type(Type_Vector):: rv
 real(R_P):: re
 type(Type_Conservative):: cons_scal,cons_1D(10),cons_2D(10,2),cons_3D(10,2,3),cons_4D(10,2,3,4)
 ...
 ! setting cons_scal, cons_1D, cons_2D, cons_3D  and cons_4D
 call set(rs,rv,re,Ns,cons_scal)
 call set(rs,rv,re,Ns,cons_1D)
 call set(rs,rv,re,Ns,cons_2D)
 call set(rs,rv,re,Ns,cons_3D)
 call set(rs,rv,re,Ns,cons_4D)
 ...
Note:
rs,rv,re,Ns are optional.
