elemental function data_type_tensor::transpose::transpose_ten ( type(type_tensor), intent(in)  ten)
private

Function for transposing a tensor.

The transpositions is done as following:

|x%x x%y x%z| |x%x y%x z%x|
ten = |y%x y%y y%z| => tran = |x%y y%y z%y|
|z%x z%y z%z| |x%z y%z z%z|
Parameters
[in]tenTensor to be transposed.
