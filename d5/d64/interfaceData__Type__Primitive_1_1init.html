Detailed Description

Subroutine for initializing Type_Primitive variable.

It is a generic interface to 4 different subroutines as it can be used for initializing scalar variables, 1D/2D or 3D arrays. The calling signatures are: 

 ...
 integer(I_P):: Ns
 real(R_P):: r(1:Ns)
 type(Type_Vector):: v
 real(R_P):: p,d,g
 type(Type_Primitive):: prim_scal,prim_1D(10),prim_2D(10,2),prim_3D(10,2,3)
 ...
 ! initializing prim_scal, prim_1D, prim_2D and prim_3D
 call init(r,v,p,d,g,Ns,prim_scal)
 call init(r,v,p,d,g,Ns,prim_1D)
 call init(r,v,p,d,g,Ns,prim_2D)
 call init(r,v,p,d,g,Ns,prim_3D)
 ...
Note:
r,v,p,d,g,Ns are optional.
