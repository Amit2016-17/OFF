type data_type_hashid::type_hashid

Derived type containing the definition of the unique key, ID, used within the hash tables.

Note
The identifier is a function of the form ID=(blk,bcl,lvl,pth) where:
'blk' are the bits identifying block;
'bcl' are the bits identifying the(parent) base cell;
'lvl' are the bits identifying the level;
'pth' are the bits describing the path(of children).
The range of ID keys can be computed as following. Each base cell can contain a maximum of children cells. As a consequence the maximum number of cells are where Nb is the (maximum) number of blocks, Nbc is the (maximum) number of the base cells.
