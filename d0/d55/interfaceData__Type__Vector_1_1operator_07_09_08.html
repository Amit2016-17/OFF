Detailed Description

Sum operator (+) overloading.

Note:
The admissible summations are:
  • Type_Vector + Type_Vector: each component of first Vector variable (vec1) is summed with the corresponding component of the second one (vec2), i.e.



  • scalar number (real or integer of any kinds defined in IR_Precision module) + Type_Vector: each component of Type_Vector is summed with the scalar, i.e.
