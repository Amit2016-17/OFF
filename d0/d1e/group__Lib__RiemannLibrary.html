module lib_riemann

This module contains procedures for computing the solution of the Riemann Problem for the Euler's conservation laws.

This is a library module. The Riemann Problem solvers provide as solution the convective fluxes normal to the interface direction. The solvers contained into this library have a unique API. They take as input the primitive variables in the left (state 1) and right (state 4) cells with respect the interface and provide the convective fluxes as output:

...
call riem_solver(p1,r1,u1,g1,p4,r4,u4,g4,f_r,f_u,f_e)
...

where define the left state, define the right state and , and are the convective fluxes of mass, momentum and energy conservation, respectively.
