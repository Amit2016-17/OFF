Detailed Description

This module contains the definition of Type_Primitive and its procedures.

Type_Primitive is a derived type that handles primitive fluid dynamic variables.

Note
The operators of assignment (=), multiplication (*), division (/), sum (+) and subtraction (-) have been overloaded. Therefore this module provides a far-complete algebra based on Type_Primitive derived type. This algebra simplifies the numerical integration of Partial Differential Equations (PDE) systems based on primitive formulation.
