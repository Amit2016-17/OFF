Detailed Description

Not-equal-to boolean operator (/=) overloading.

Note:
The boolean comparison between two vectors is made on normL2 and direction, while the comparison between scalar number (real or integer of any kinds as defined in IR_Precision module) is made on only normL2 of the vector.
