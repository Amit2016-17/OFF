type Data_Type_BC::Type_BC

Derived type containing boundary conditions informations.

Note:
meaning of inf component
  • bc_in1 type condition: supersonic inflow steady conditions
    the component inf is the index array of inflow boundary conditions; inflow 1 conditions (primitive variables) are stored in array "in1" so the boundary conditions can be accessed by: 
     in1(1:Np,inf)
  • bc_in2 type condition: supersonic inflow unsteady conditions
    the component inf is the index array of inflow boundary conditions; inflow 2 conditions (primitive variables) are stored in array "in2" so the boundary conditions can be accessed by: 
     in2(1:Np,n,inf)
    where n is the time step counter.

type Data_Type_Conservative::Type_Conservative

Derived type containing conservative variables.

Note:
This derived type can represent multi species fluids. The density component, rs, is a dynamic memory component defined as an allocatable 1D array. rs is allocated at runtime with the number of initial species that constitute the initial fluid mixture. Due to the presence of a dynamic component a freeing memory "method" for this component is necessary. Before deallocate a variable defined as Type_Conservative the free function must be invoked to free the memory of the dynamic component.
