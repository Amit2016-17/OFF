Detailed Description

Read overloading of Type_Probe variable.

This is a generic interface to 2 functions (one binary and another ascii) for reading scalar variables. The functions return an error integer code. The calling signatures are: 

 ...
 integer(I4P):: err,unit
 character(1):: format="*"
 type(Type_Probe):: probe
 ...
 ! formatted reading
 err = read(unit,format,probe)
 ! binary reading
 err = read(unit,probe)
 ...
