The Single Linked List data structure is used to resolve collisions into the hash table of the cells data.

type data_type_hashtface::type_hashtface

Derived type containing the hash table of cells data.

The the collisions of different (unique) IDs into the same hash buckets are resolved by the "chaining" technique by means of a single linked list data structure.
