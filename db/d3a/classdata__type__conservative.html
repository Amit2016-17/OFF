Detailed Description

This module contains the definition of Type_Conservative and its procedures.

Type_Conservative is a derived type that handles conservative fluid dynamic variables.

Note
The operators of assignment (=), multiplication (*), division (/), sum (+) and subtraction (-) have been overloaded. Therefore this module provides a far-complete algebra based on Type_Conservative derived type. This algebra simplifies the numerical integration of Partial Differential Equations (PDE) systems based on conservative formulation.
