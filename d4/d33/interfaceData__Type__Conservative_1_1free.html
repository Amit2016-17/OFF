Detailed Description

Function for freeing the memory of Type_Conservative dynamic components.

This is a generic interface to 5 functions as it can be used for scalar variables, 1D/2D/3D or 4D arrays. The calling signatures are: 

 ...
 integer(I4P):: err
 type(Type_Conservative):: cons_scal,cons_1D(10),cons_2D(10,2),cons_3D(10,2,3),cons_4D(10,2,3,4)
 ...
 ! freeing dynamic components memory of cons_scal, cons_1D, cons_2D, cons_3D  and cons_4D
 err = free(cons_scal)
 err = free(cons_1D)
 err = free(cons_2D)
