Detailed Description

Multiplication operator (*) overloading.

Note:
The admissible multiplications are:
  • Type_Tensor * Type_Tensor: each component of first tensor variable (ten1) is multiplied for the corresponding component of the second one (ten2), i.e.



  • scalar number (real or integer of any kinds defined in IR_Precision module) * Type_Tensor: each component of Type_Tensor is multiplied for the scalar, i.e.
