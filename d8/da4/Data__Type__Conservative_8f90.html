type Data_Type_Conservative::Type_Conservative

Derived type containing conservative variables.

Note:
This derived type can represent multi species fluids. The density component, rs, is a dynamic memory component defined as an allocatable 1D array. rs is allocated at runtime with the number of initial species that constitute the initial fluid mixture. Due to the presence of a dynamic component a freeing memory "method" for this component is necessary. Before deallocate a variable defined as Type_Conservative the free function must be invoked to free the memory of the dynamic component.
