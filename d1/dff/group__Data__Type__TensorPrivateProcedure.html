elemental subroutine Data_Type_Tensor::transpose_self ( class(Type_Tensor), intent(inout)  ten) [private]

Subroutine for transposing a tensor.

The transpositions is done as following: 

       |x%x x%y x%z|           |x%x y%x z%x|
 ten = |y%x y%y y%z| => tran = |x%y y%y z%y|
       |z%x z%y z%z|           |x%z y%z z%z|
Parameters:
[in,out]tenTensor to be transposed.
