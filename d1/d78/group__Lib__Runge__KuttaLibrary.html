module lib_runge_kutta

This module contains the definition of Runge-Kutta procedures for time integration.

Note
This is a library module. This module provides procedure for performing ODE time integration by means numerical scheme belongs to the explicit, Strong Stability Preserving Runge-Kutta family. The implementation is explicit without low-storage form. The scheme is written using the Butcher table (see Butcher, J.C., "Coefficients for the study of Runge-Kutta integration processes", J.
